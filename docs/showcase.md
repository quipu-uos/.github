# QUIPU Projects Showcase

## 1) QUIPU Main Web (2024)

### Overview
QUIPU의 2024 공식 웹사이트로, 동아리의 첫인상을 전달하는 메인 채널 역할을 담당한 프로젝트입니다. 동아리 소개, 모집 안내, 활동 기록, 프로젝트 쇼케이스를 하나의 흐름으로 구성해 방문자가 필요한 정보를 빠르게 찾을 수 있도록 설계했습니다.

### Period
- 2024.01 ~ 2024.03 (Main Web)
- 2024.08 (Roulette Event)

### Tech Stack
- Frontend: React
- Backend: Express
- Database: MySQL

### Links
- Service: <https://quipu-main-2024.vercel.app>
- GitHub: <https://github.com/quipu-uos/main-2024>

### Highlights
- 동아리 소개, 모집, 활동 기록을 메인 웹에서 통합 제공
- 프로젝트 쇼케이스 페이지를 통해 결과물을 한눈에 확인 가능
- 이벤트 기간에는 룰렛 기능을 추가해 참여형 경험 확장

## 2) 배틀글라운드 (2024)

### Overview
서울시립대학교 2024 인향제 축제 부스에서 운영한 실시간 멀티플레이 타자 게임입니다. 총 70명이 참여했으며, 현장에서 즉시 이해하고 즐길 수 있는 경쟁형 콘텐츠로 동아리 체험의 몰입도를 높였습니다.

### Period
- 2024.07 ~ 2024.08

### Tech Stack
- Frontend: React
- Backend: Express, Socket.IO
- Database: MySQL

### Links
- GitHub: <https://github.com/quipu-uos/battleground>

### Highlights
- 방 생성/참가 방식으로 3인이 함께 플레이하는 구조
- 60초 동안 단어를 먼저 입력해 점수를 획득하는 경쟁 규칙
- 점수와 순위를 실시간으로 확인할 수 있는 게임 화면 제공
- 게임 종료 후 방 순위와 전체 TOP 10 랭킹 제공

## 3) LingoConnect

### Overview
2024 새싹 해커톤 출품작으로, 생성형 AI 음성 대화를 활용해 대화 흐름과 발음을 함께 연습할 수 있도록 설계한 언어 학습 서비스입니다. 단순 평가를 넘어, 반복 학습과 표현 훈련까지 이어지는 경험을 만드는 데 초점을 맞췄습니다.

### Period
- 2024.07 ~ 2024.08

### Tech Stack
- Frontend: React, JavaScript
- Backend: Spring, Java
- AI/Voice: 생성형 AI 기반 대화·피드백

### Links
- GitHub: <https://github.com/i2na/LingoConnect>

### Highlights
- AI와의 음성 대화로 일상 소통 상황을 자연스럽게 연습
- 대화 흐름 피드백과 발음 점수(5점 척도) 제공
- 이미지 묘사 기반 표현 학습으로 언어 확장 훈련 지원
- 퀴즈·보상 요소를 통한 학습 동기 강화

## 4) WaggleWaggle

### Period
- 2024.11 (Git 이력 기준)

### Tech Stack
- Frontend: React, JavaScript
- Backend: Spring Boot
- Realtime/AI: WebSocket, OpenAI Assistant API

### Links
- GitHub: <https://github.com/i2na/wagglewaggle>

### Highlights
- 위치 기반 실시간 혼잡도 분석 및 시각화 제공
- 관리자가 지도에서 폴리곤 구역을 지정해 운영 구역 관리
- 실시간 공지 및 긴급 알림 전달 기능 제공
- 참가자 간 채팅과 정보 공유를 통한 현장 커뮤니케이션 지원

## 5) Articket

### Overview
CO-SHOW 차세대통신 경진대회 COSS 협의회장상 수상작입니다. 사용자가 업로드한 사진을 화가 스타일로 변환해 티켓 형태로 출력하는 체험형 서비스로, 2025 서울시립대학교 대동제 동아리 부스에서도 운영했습니다.

### Period
- 2024.09 ~ 2024.11 (Articket 1.0)
- 2025.02 (Articket 2.0)

### Tech Stack
- Frontend: React, WebSocket
- Backend: Flask, WebSocket
- AI/Infra: Stable Diffusion API, RunPod GPU, Railway, Netlify

### Links
- GitHub: <https://github.com/i2na/Articket>

### Highlights
- 성격 테스트 기반 화가 스타일 추천(피카소, 르누아르, 고흐, 리히텐슈타인)
- 업로드한 사진을 화풍 이미지로 변환해 결과 제공
- 스마트폰-모니터 연동 방식으로 부스 체험 흐름 구성
- 변환 결과를 포함한 티켓 PDF 자동 생성·출력 지원
